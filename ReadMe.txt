The framework uses Cypress and is aligned to Page Object Model. 
It also uses mocha framework and chai assertions.

prerequisite:
1. Node.js should be installed.
2. Editor used - VS Code

cypress/fixture : This folders stores the inout data in json format which are used in the script
cypress/integration/examples : This folder contains the src code.

Project Structure :
1.invia_suite.js is the src code. It contains all the test cases mentioned in the task.
2. There are 3 POM files : Homepage.js, HotelauswahlPage.js and Hoteldetails.js
